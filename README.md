# vitacare_hostinger

VitaCare 🌿
VitaCare is a web-based application designed to help users track their daily vitamin intake and maintain a healthy routine.

✨ Features
Vitamin Tracking: Easily record and view your daily nutrient history.

Streak System: Stay motivated by tracking your consecutive days of vitamin intake.

Admin Dashboard: A centralized dashboard for managing the system, featuring a Vitamins Table for data monitoring and updates.

Secure Access: User authentication and session management to keep data private.

🛠️ Tech Stack
Backend: PHP

Database: MySQL 

Frontend: HTML, CSS, and JavaScript

Hosting: Hostinger
