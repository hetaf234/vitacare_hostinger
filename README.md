# vitacare_hostinger

VitaCare is a web-based platform designed to help users manage their vitamin intake and maintain a healthier lifestyle more efficiently. The system provides an intuitive interface that allows users to register, log in, and organize their personalized vitamin schedules with ease. Through features such as adding, editing, and deleting vitamins, as well as receiving timely reminders and tracking intake history, VitaCare simplifies daily supplement management while offering a clean, user-friendly, and health-focused experience.

Tech Stack
Backend: PHP

Database: MySQL 

Frontend: HTML, CSS, and JavaScript

Hosting: Hostinger
